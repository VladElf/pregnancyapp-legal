Privacy Policy

Effective date: July 2026

Information collected

The application stores pregnancy information locally on your device using SQLite.

  • Last menstrual period
  • Weight records
  • Kick counter
  • Reminders
  • Application preferences

Analytics

PregnancyApp uses Firebase Analytics to collect anonymous usage statistics. These data help improve the application and do not directly identify users.

Advertising

The free version may display ads using Google AdMob.

Future Premium Features

Future versions may offer optional premium subscriptions.

Notifications

Notification permission is used only for pregnancy reminders.

Data deletion

You can delete all local data from the Settings screen or by uninstalling the application.
